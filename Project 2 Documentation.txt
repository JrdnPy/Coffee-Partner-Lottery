3. Running Instructions

To run the Homies Meet Up program:

Prerequisites:
- Python 3.x installed (Anaconda recommended)
- The file `credentials.json` must be placed in the project folder (this file is excluded from GitHub for security reasons – ask any team member for it)

Steps to run:

1. Download or clone the project folder containing CoffeePairing.py, Conversation_starters.csv, and credentials.json.

2. Make sure `credentials.json` is in the exact same folder as CoffeePairing.py.

3. Open a terminal/command prompt and navigate to the project folder.

4. Run the program with the following command:
   python CoffeePairing.py or click on the programme to run.

The program will automatically:
- Connect to our live Google Form/Sheet via the API
- Read the current participants
- Ask for the desired group size
- Generate random groups
- Print the groups on screen
- Create a “messages” folder with personalised messages for every group

For testing/grading purposes, you can freely add, edit, or delete participants:
PLEASE NOTE: Participants AND their respective indexed cells must be deleted from both forms and sheets !

Please Note:  

The program includes an auto-installer that will check for and install the following required packages via `pip` if they are missing:
* `pandas` (for data manipulation)
* `gspread` (for Google Sheets API integration)

Because we use direct API access, there is no need to manually download any CSV file.
